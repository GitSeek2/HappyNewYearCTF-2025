# HappyNewYearCTF 2025

HappyNewYearCTF 2025 的题目源码仓库。

HappyNewYearCTF 2025 是一套用于团队内练习和一些教学活动的题目集，部署在 [CSSEC::CTF](https://ctf.cssec.cc) 平台。
